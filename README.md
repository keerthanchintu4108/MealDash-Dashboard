# 🍽️ Meal Dashboard – Power BI

## 📌 About
The **Meal Dashboard** is an interactive Power BI dashboard created to analyze food order data. It helps visualize order trends, customer behavior, and popular food items using dynamic charts and key performance indicators (KPIs). The dashboard provides clear insights that can help understand customer preferences and improve decision making.

## 📊 Features
- Interactive Power BI dashboard  
- Analysis of food order trends  
- Insights into customer behavior  
- Identification of most popular food items  
- Dynamic filters and slicers  
- Clear visualizations and KPI indicators  

## 📈 Key Metrics
- Total Orders  
- Total Revenue  
- Most Ordered Food Items  
- Order Trends Over Time  
- Customer Ordering Patterns  

## 🛠️ Tools Used
- Power BI  
- Data Visualization  
- Data Analysis
